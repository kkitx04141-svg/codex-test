# React 学习笔记

## React 是什么

React 是一种“用来做网页界面”的工具。

你可以把网页理解成一个房间，里面有按钮、文字、图片、输入框这些“家具”。React 的作用，就是帮开发者更方便地搭建这些家具，并且在数据变化时，自动把界面更新好。

更简单一点说：

- 普通网页开发：你要自己一点点改页面内容
- React：你只需要告诉它“现在数据是什么”，它来帮你把页面显示成对应样子

## 一个简单例子

比如一个计数器：

- 数据是 `3`
- 页面就显示“当前数字：3”
- 你点一下按钮，数据变成 `4`
- React 会自动把页面改成“当前数字：4”

## React 为什么受欢迎

- 适合做交互很多的网站和应用
- 页面拆分清楚，方便维护
- 组件化很强

## 什么是组件

“组件”是 React 里很重要的概念。你可以把组件理解成“可重复使用的小积木”，比如：

- 一个按钮组件
- 一个导航栏组件
- 一个商品卡片组件

把这些积木拼起来，就能做出完整网站。

## 一句话总结

React = 用“组件 + 数据驱动”的方式来做网页界面的工具。
